Удалите этот файл из репозитория GitHub, чтобы сбросить кэш изображений на всех устройствах.
При удалении этого файла сервер начнет выдавать уникальные метки времени при каждом запросе, заставляя браузеры всех посетителей мгновенно загрузить самые свежие изображения.
Чтобы вернуть обычное кэширование, просто создайте этот файл обратно в корневой папке репозитория.
